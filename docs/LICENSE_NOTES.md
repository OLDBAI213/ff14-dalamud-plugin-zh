# License Notes

This repository contains modified distribution packages for third-party Dalamud plugins.

Current status: license and redistribution rights have not been audited plugin-by-plugin. Before promoting this as a public plugin repository, check every upstream plugin license and remove any package that does not permit redistribution of modified binaries.

Translation maps and notes are maintained here so the localization work can be reproduced against newer plugin versions instead of permanently depending on binary redistribution.
