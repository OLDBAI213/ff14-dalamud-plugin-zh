# FF14 Dalamud Plugin Chinese Localization

This repository stores localized package metadata, release packages, and translation maps for selected XIVLauncherCN / Dalamud plugins.

Plugin repo URL for XIVLauncherCN / Dalamud custom repositories:

```text
https://raw.githubusercontent.com/OLDBAI213/ff14-dalamud-plugin-zh/main/pluginmaster.json
```

Current release:

```text
https://github.com/OLDBAI213/ff14-dalamud-plugin-zh/releases/tag/localized-2026-07-09
```

See `docs/LOCALIZATION_STATUS.md` for status and verification notes.

Close FF14, XIVLauncher, and Dalamud before replacing installed plugin files manually. XIVLauncher/Dalamud plugin updates may overwrite these localized DLLs or resource DLLs.

License and redistribution rights have not been audited plugin-by-plugin. Treat this repository as a personal localization mirror unless each upstream plugin license is checked.
