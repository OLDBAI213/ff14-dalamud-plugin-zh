## Notes

These packages were generated from a local XIVLauncherCN installation on 2026-07-09. Close FF14, XIVLauncher, and Dalamud before replacing installed plugin files manually. XIVLauncher/Dalamud plugin updates may overwrite these localized DLLs or resource DLLs.

License and redistribution rights have not been audited plugin-by-plugin. Treat this repository as a personal localization mirror unless each upstream plugin license is checked.
